🛒 Mini Market Game

Console mini-market simulator with account system, persistent balance, inventory management and built-in Casino (Blackjack).

⸻

📌 About The Project

Mini Market — это консольная игра на C++, в которой игрок может:
 • Создать аккаунт
 • Войти в существующий аккаунт
 • Покупать товары
 • Продавать товары
 • Управлять своим инвентарём
 • Играть в казино (Blackjack)
 • Сохранять прогресс между запусками программы

Все данные пользователя сохраняются в .txt файлах, поэтому игра не сбрасывается после закрытия.

⸻

🚀 Features
 • 🔐 Регистрация и авторизация
 • 💰 Система баланса

 • 📦 Инвентарь игрока

🧠 How It Works

При запуске программа:
 1. Проверяет наличие файла products.txt
 2. Создаёт магазин, если его нет
 3. Предлагает регистрацию или вход
 4. Загружает:
 • пароль
 • баланс
 • инвентарь

Каждый пользователь имеет:

login.txt
login_balance.txt
login_inventory.txt


🎮 Game Menu

После входа доступны команды:

menu - открыть меню
exit - выйти

Menu:
1 - Магазин
2 - Инвентарь
3 - Купить
4 - Продать
5 - Баланс
6 - Казино
0 - Выход


🎰 Casino Mode
Реализована упрощённая версия BlackJack:
 • Игрок делает ставку
 • Получает карты
 • Может брать ещё или остановиться
 • Бот играет по своей логике
 • Баланс обновляется автоматически
